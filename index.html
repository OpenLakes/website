A complete lakehouse in one stack

Battle-tested open-source projects, wired together with sane defaults.

Storage & Tables

MinIO for S3-compatible object storage. Apache Iceberg for ACID transactions, time travel, and schema evolution. Project Nessie for Git-like version control of your data.

Query & Compute

Trino for fast interactive SQL. Apache Spark for batch processing and streaming. Both engines share the same Iceberg catalog and storage layer.

Orchestration

Apache Airflow coordinates your pipelines with the KubernetesExecutor. Schedule notebooks, dbt models, Spark jobs, and SQL transformations from one place.

Ingestion

Meltano with Singer taps for batch ELT from 500+ sources. Debezium for real-time CDC from PostgreSQL, MySQL, and MongoDB to Kafka and Iceberg.

Analytics & BI

Apache Superset for self-service dashboards connected to Trino. JupyterHub for collaborative notebooks with Spark kernels pre-configured.

Catalog & Governance

OpenMetadata for data discovery, lineage tracking, and data quality. Automatically catalog all your Iceberg tables and pipeline metadata.
